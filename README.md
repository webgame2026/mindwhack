1.  **GitHub Repo**: Create a new public repository.
2.  **Upload Files**: Drag and drop all project files into your repo. **Make sure to include `package.json` and the `.github` folder!**
3.  **Add API Key (Optional but Recommended)**:
    - Go to your repo on GitHub.
    - Click **Settings** > **Secrets and variables** > **Actions**.
    - Click **New repository secret**.
    - Name: `GEMINI_API_KEY`
    - Value: Paste your Google Gemini API Key here.
4.  **GitHub Pages**: 
    - Go to **Settings > Pages**.
    - Set **Source** to **GitHub Actions**.
5.  **Watch the Build**:
    - Click the **Actions** tab.
    - Wait for the "Deploy to GitHub Pages" task to finish (Green Checkmark).
    - Your link will appear there!
